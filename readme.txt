An immersive, pixel-perfect Elementor addon plugin bringing 3D Scroll-Bound Canvas animation, telemetry HUDs, specification Bento Grids, and interactive Reservation CTAs into WordPress.

== Description ==

This plugin provides four professional, high-performance Elementor widgets modeled exactly after the Kami K1 3D web experience:
* **Kami K1 Navbar:** Sticky dark desktop/mobile header navigation bar with accent color glow and animated sliding CTA button.
* **Kami K1 3D Scrollytelling:** Full viewport scrolling canvas scrubbing system featuring 192 frame animations, CAD targets, vertical dot progress HUD navigation, telemetry overlays, diagnostic widgets, and 9 fade-in narrative blocks.
* **Kami K1 Bento Grid:** Clean specifications dashboard grid showcasing OLED simulated display widget, CNC chassis, Poron isolator details, and hot-swap keycaps.
* **Kami K1 Reservation CTA:** batch certification specs, custom edition selection, switch type selection, name/email input, terminal transmission simulation log console, and reservation success state.

== Installation ==

1. Upload the entire `kami-k1-elementor-addon` directory to your WordPress site's `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open any Page with Elementor.
4. Search for widgets under the "Kami K1 Widgets" category in the left panel.
5. Drag and drop the widgets into your Elementor canvas!

== Frequently Asked Questions ==

= Where are the frame images stored? =
The frame images are stored in `/assets/frames/` directory of this plugin. The scrollytelling javascript handles loading and drawing them onto the canvas element automatically.

= Do I need to write custom CSS? =
No, the widgets are fully self-contained and pre-styled to match the dark, technical, red-accented aesthetics of the original Next.js mechanical keyboard site.
